## 레이아웃 구조

### 2열 그리드
```
┌─────────────────────────────┬──────────┐
│ Main Content                │ Sidebar  │
│ 1fr (scrollable)            │ 320px    │
│                             │(sticky)  │
└─────────────────────────────┴──────────┘
```

### 스티키 헤더
- 제목 + 부제목
- 액션 버튼: 📤📥💾📄📊
- 날짜 필터: 오늘/7일/30일/사용자정의

### 메인 콘텐츠 (위→아래)
1. **Metrics Grid** - 8개 카드 (지표)
2. **Daily Trend Chart** - 일별 추세 (Full Width)
3. **Distribution Charts** - ROAS 분포 + 예산 분배 (2x1)
4. **Funnel & Comparison** - 전환 + 캠페인 비교 (2x1)
5. **Weekday Heatmap** - 요일별 성과 (Full Width)
6. **CREATIVE ANALYSIS ⭐** - 크리에이티브 (강조됨!)
   - 4개 탭: Top ROAS | Top CVR | All | Low Performers
7. **Campaign Table** - 캠페인별 성과 (Full Width)
8. **AI Insights** - AI 분석 결과 (Full Width)

### 오른쪽 사이드바
1. **Saved Analyses** - 저장된 분석 목록
2. **Period Comparison** - 기간 비교 도구
3. **Goals & Budget** - 목표 설정 및 예산 추적

## 크리에이티브 분석 강조

### 시각적 강조 방법
✅ 파란색 2px 테두리  
✅ 밝은 그라데이션 배경  
✅ "⭐ 핵심 분석" 배지  
✅ 다른 섹션과 분리  

### 4개 탭 제공
- **💎 Top ROAS**: 수익 성과 순위
- **🎯 Top CVR**: 전환 성과 순위
- **📋 All Creatives**: 전체 크리에이티브 목록
- **⚠️ Low Performers**: 개선 필요한 크리에이티브
