# Excel 템플릿 가이드

## 템플릿 형식

광고 데이터를 업로드하려면 다음 형식의 Excel 또는 CSV 파일이 필요합니다.

### 필수 컬럼 (Required Columns)

| 컬럼명 | 설명 | 예시 |
|--------|------|------|
| date | 날짜 (YYYY-MM-DD) | 2024-11-01 |
| campaign_name | 캠페인명 | 블랙프라이데이_신규고객 |
| spend | 지출액 (원) | 150000 |
| clicks | 클릭수 | 1200 |
| conversions | 전환수 (구매) | 48 |
| revenue | 매출액 (원) | 540000 |

### 선택 컬럼 (Optional Columns)

| 컬럼명 | 설명 | 예시 |
|--------|------|------|
| impressions | 노출수 | 45000 |

---

## 예시 데이터

### Excel/CSV 파일 예시:

```
date,campaign_name,spend,impressions,clicks,conversions,revenue
2024-11-01,블프_신규,150000,45000,1200,48,540000
2024-11-02,블프_신규,160000,48000,1300,52,580000

2024-11-01,기존고객_A,80000,20000,800,30,360000
2024-11-02,기존고객_A,85000,22000,850,32,384000
2024-11-01,리타겟팅_B,120000,35000,1000,40,480000
```

## 플랫폼별 데이터 추출 방법

### 네이버 검색광고

1. 네이버 광고 관리 시스템 로그인
2. **통계/리포트** 메뉴 선택
3. **기간 설정** (시작일~종료일)
4. **그룹 기준**: 캠페인
5. **다운로드** → Excel 형식
6. 컬럼명을 위 형식에 맞게 변경

**네이버 컬럼 매칭:**
- 일자 → date
- 캠페인명 → campaign_name
- 비용 → spend
- 노출수 → impressions
- 클릭수 → clicks
- 전환수 → conversions
- 전환매출액 → revenue

### 메타 광고 (Facebook/Instagram)

1. Meta 비즈니스 관리자 로그인
2. **광고 관리자** → **리포트**
3. **맞춤 보고서 만들기**
4. 분류 기준: **일별** + **캠페인**
5. 지표 선택:
   - 지출
   - 노출수
   - 링크 클릭수
   - 구매 전환수
   - 구매 전환 가치
6. **내보내기** → CSV

**Meta 컬럼 매칭:**
- 일 → date
- 캠페인 이름 → campaign_name
- 지출 → spend
- 노출수 → impressions
- 링크 클릭 → clicks
- 구매 → conversions
- 구매 전환 가치 → revenue

### 구글 광고 (Google Ads)

1. Google Ads 로그인
2. **보고서** → **사전 정의된 보고서(차원)** → **기본 - 시간**
3. 기간 설정 및 **캠페인** 차원 추가
4. 지표 선택:
   - 비용
   - 노출수
   - 클릭수
   - 전환수
   - 전환 가치
5. **다운로드** → CSV

**Google Ads 컬럼 매칭:**
- 일 → date
- 캠페인 → campaign_name
- 비용 → spend
- 노출수 → impressions
- 클릭수 → clicks
- 전환수 → conversions
- 전환 가치 → revenue

### 카카오모먼트

1. 카카오모먼트 로그인
2. **통계** → **캠페인 통계**
3. 기간 설정
4. **엑셀 다운로드**

**카카오모먼트 컬럼 매칭:**
- 날짜 → date
- 캠페인명 → campaign_name
- 소진금액 → spend
- 노출수 → impressions
- 클릭수 → clicks
- 전환수 → conversions
- 전환매출 → revenue

## 템플릿 생성 방법 (Excel)

### 방법 1: Excel에서 직접 생성

1. Excel 새 파일 생성
2. 첫 번째 행에 컬럼명 입력:
   ```
   date | campaign_name | spend | impressions | clicks | conversions | revenue
   ```
3. 두 번째 행부터 데이터 입력
4. **.xlsx** 형식으로 저장

### 방법 2: CSV 파일 생성

1. 메모장이나 텍스트 편집기 열기
2. 첫 줄에 헤더 작성:
   ```
   date,campaign_name,spend,impressions,clicks,conversions,revenue
   ```
3. 데이터 행 추가 (쉼표로 구분)
4. **.csv** 형식으로 저장 (UTF-8 인코딩)

## 주의사항

### ✅ DO (권장사항)

- **날짜 형식**: YYYY-MM-DD (예: 2024-11-01)
- **캠페인명**: 한글/영문/숫자/언더스코어 사용 가능
- **숫자**: 천 단위 구분 없이 입력 (150000, not 150,000)
- **소수점**: 불필요 (정수만 입력)
- **파일 인코딩**: UTF-8 (한글 지원)

### ❌ DON'T (주의사항)

- **날짜 형식 틀림**: 11/01/2024, 2024.11.01 ❌
- **통화 기호**: ₩, $ 등 기호 사용 금지
- **쉼표**: 숫자에 천 단위 쉼표(150,000) 사용 금지
- **빈 값**: 필수 컬럼은 반드시 입력
- **중복 헤더**: 여러 헤더 행 사용 금지

---

## 파일 크기 제한

- **최대 파일 크기**: 10MB
- **최대 행 수**: 제한 없음 (단, 성능상 10,000 행 이하 권장)

## 업로드 오류 해결

### 오류 1: "필수 컬럼이 누락되었습니다"

**원인**: date, campaign_name, spend, clicks, conversions, revenue 중 하나라도 없음

**해결**: 모든 필수 컬럼이 첫 번째 행에 정확히 입력되어 있는지 확인

### 오류 2: "파일 형식이 올바르지 않습니다"

**원인**: .xlsx, .xls, .csv 이외의 파일 형식

**해결**: Excel 또는 CSV 형식으로 다시 저장

### 오류 3: "날짜 형식이 잘못되었습니다"

**원인**: date 컬럼이 YYYY-MM-DD 형식이 아님

**해결**: Excel에서 date 컬럼을 선택 → 셀 서식 → 날짜 → YYYY-MM-DD 형식 선택

### 오류 4: "숫자 형식이 잘못되었습니다"

**원인**: spend, clicks 등에 문자가 포함되거나 쉼표(,) 사용

**해결**: 숫자 컬럼에서 통화 기호($, ₩) 및 쉼표 제거
